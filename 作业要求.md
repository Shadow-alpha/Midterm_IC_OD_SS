## 🟢 任务 1：卷积神经网络微调与宠物（花卉）识别

### 1. 实验目标

在 ImageNet 预训练模型的基础上，通过微调（Fine-tuning）实现对 102 类花卉的精准识别，并探究注意力机制或新架构的提升效果。

### 2. 基本要求

- **数据集**：[102 Category Flower Dataset](https://www.robots.ox.ac.uk/~vgg/data/flowers/102/)。
- **Baseline 构建**：
  - 修改 **ResNet-18** 或 **ResNet-34** 的输出层。
  - **初始化策略**：加载 ImageNet 预训练参数。
  - **训练策略**：输出层从零开始训练；其余预训练参数采用 **较小的学习率** 进行微调。
- **超参数分析**：
  - 探究训练步数（Epochs/Iterations）、学习率（LR）及其不同组合对模型性能的影响。
- **预训练消融实验**：
  - 对比“预训练权重微调”与“随机初始化从零训练”在相同数据集上的表现差异。
- **模型改进（引入注意力机制）**：
  - **方式 A**：手动修改网络结构，加入注意力模块（如 **SE-block**, **CBAM** 等）。
  - **方式 B**：直接改用轻量级 Vision Transformer（如 **ViT-Tiny**, **Swin-T**）。
  - **对比要求**：将改进后的模型性能（Accuracy）与 Baseline 进行对比。

## 🔵 任务 2：场景目标检测与视频多目标跟踪

### 1. 实验目标

微调现代单阶段检测模型，并结合跟踪算法解决实际场景中的目标统计与行为分析问题。

### 2. 基本要求

- **数据集**：[Road Vehicle Images Dataset](https://www.kaggle.com/datasets/sshikamaru/road-vehicle-images-dataset)。
- **模型微调**：使用 **YOLOv8** 或同类单阶段模型，训练得到针对车辆的专属检测模型。
- **视频流检测与跟踪**：
  - **测试视频**：准备一段 **10-30 秒** 的自拍视频（如校园、路口等场景）。
  - **跟踪逻辑**：结合多目标跟踪算法（如 YOLOv8 内置 tracking），输出 Bounding Box、类别及 **稳定的 Tracking ID**。
- **遮挡与 ID 跳变分析（核心挑战）**：
  - 在视频中寻找目标遮挡或密集交汇的片段。
  - **可视化展示**：截取连续 **3-4 帧** 画面进行对比。
  - **理论分析**：结合课上理论，详细分析算法是成功维持了 ID 还是发生了丢失/跳变，并阐述底层原理。
- **越线计数功能**：
  - 在画面中设定一条 **虚拟线**。
  - **逻辑实现**：利用检测框中心坐标与 Tracking ID 的连续性，统计并实时显示跨越该线的物体总数。

## 🟡 任务 3：U-Net 从零搭建与损失函数工程

### 1. 实验目标

深入理解语义分割的网络结构，并通过手动实现损失函数解决像素不平衡问题。

### 2. 基本要求

- **网络搭建（禁止使用预训练权重）**：
  - 使用基础 API **从零手写搭建 U-Net**。
  - 结构必须包含：完整的下采样编码器、上采样解码器、以及 **Skip Connection（特征拼接）** 逻辑。
- **轻量级数据集训练**：
  - 使用 [Stanford Background Dataset](http://dags.stanford.edu/projects/scenedataset.html)。
  - **初始化**：必须使用随机初始化的权重开始训练。
- **损失函数工程（Dice Loss）**：
  - **手动代码实现 Dice Loss**。
  - **对比实验**：分别使用以下三种配置训练模型，对比验证集上的 **mIoU** 表现：
    1. 仅使用标准交叉熵损失（Cross-Entropy Loss）。
    2. 仅使用手动实现的 Dice Loss。
    3. 组合损失（Cross-Entropy + Dice Loss）。
